🚀 TradeKaro Deployment Guide

✅ Your TradeKaro is Ready to Deploy!

Live URL: https://tradekaro-india.surge.sh

Step 1: Open Command Prompt

Press Win + R, type cmd, press Enter

Step 2: Navigate to Project

cd c:\Users\samurai\Downloads\tradeKAro\tradeKAro\stocksim-india

Step 3: Install Surge

npm install -g surge

Step 4: Install Dependencies

npm install --legacy-peer-deps

Step 5: Build Project

npm run build

Step 6: Deploy to Surge

surge ./out tradekaro-india.surge.sh

When prompted, create a Surge account with your email
